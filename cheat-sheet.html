CSS Cheat Sheet

Selectors

CSS selectors allow you to select and manipulate HTML elements.

Class

Selects all elements within a class.

Child

A child selector in CSS is the "greater than" symbol(>). Example: ul>li -this would select elements that are direct descendants of ul

Hover

Selects elements when the mouse is over them

Positioning

Fixed

Will not move even if the window is scrolled

Float

Element can be pushed to the left or right, allowing other elements to wrap around it.

Z-index

Specifies the stack order of an element

Text

font-family

Changes the font-style

font-size

Changes the font size

color

Changes the color of the text -

font-style

Allows for effects such as bold and italic

text-decorations

Allows for effects such as underline
