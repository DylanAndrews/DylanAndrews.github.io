The Greatness of Git

The subtitle, if you want it

4/30/15

Version control systems are simply systems that keep track of changes, and this is extremely important in the field of web development, where one mistake has the potential do undo hours of work. In addition to tracking changes, version control allows multiple users, by way of the merging and branching systems, to work on the same project simultaneously without the risk of accidentally deleting someone else's work.

Git, which is one of the most common version systems used in web development, is at first a bit cumbersome to manage. However, once you get the hang of it you realize that it is a powerful tool that provides a safeguard against catastophic error, and a platform for efficient workflow.
